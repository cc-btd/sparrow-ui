可以折叠/展开的内容区域。
## 何时使用
- 对复杂区域进行分组和隐藏，保持页面的整洁。
- 手风琴是一种特殊的折叠面板，只允许单个内容区域展开。

## API
| 参数 | 说明 | 类型 | 可选 | 默认值 |
| ---- | ---- | ---- | ---- | ------ |
no-boder|	不显示边框，建议在灰色背景下使用| boolean|-|	false
dis-hover|	禁用鼠标悬停显示阴影|boolean|-|	false
title|	卡片标题|string|-|-
extra|	卡片额外操作的内容|	-|-|-
