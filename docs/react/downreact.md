## down 
1. 你可以通过 `地址`去下载
2. 你可以自定义所下载的项， 比如勾选下列的模块。 


```jsx
import React from 'react'

const Down = () => {

    return <div>
    
    选项</div>
}

ReactDOM.render(<Down />, mountNode);

```
